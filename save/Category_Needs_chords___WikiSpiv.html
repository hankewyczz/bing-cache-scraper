Category: Needs chords

From WikiSpiv
Jump to: navigation, search

Songs that need guitar chords to be added. This should only include songs that need them, which is not the same as all songs that don't have them. For example, most religious songs don't use instruments and thus don't need chords. Furthermore, an obscure таборова пісня does not need them, although more popular таборові пісні, like Безмежне життя, do.
